# Système Domestique Embarqué

Ce projet permet de gérer, surveiller et automatiser les équipements d'un domicile grâce à un système embarqué. Il offre une interface simple d'utilisation pour le contrôle, la sécurité et l'efficacité énergétique.

## Fonctionnalités

- Contrôle à distance des appareils
- Surveillance en temps réel
- Automatisation des tâches courantes
